# Dice_Simulator
Pequeño proyecto de práctica en Python para reforzar manejo de funciones, validación de entradas, diccionarios y control de flujo. Simulador de dados con diferentes números de caras.


A simple dice simulator built during my Python learning process.
This project helped me practice core programming concepts such as:

1. Functions and modular design

2. Input validation using try/except

3. Dictionaries used as configuration menus

4. Program flow control with loops and condition checking

5. Returning structured data (using tuples)

6. Handling user interaction in the console

#Features

Launch predefined dice: d4, d6, d20

Create custom dice with any number of sides

Validate user inputs and handle errors

Allows launching multiple dice in one execution
